:::caution

This release has moved away from parameters all being global. What this means is parameters are relevant to the command they are used with.

This basically means put the parameters after the command (`b`, `d`, `if`, etc.) instead of before.
:::

- [[#3857](https://github.com/iBotPeaches/Apktool/pull/3857)] Add support for API 36 (Baklava).
- [[#3865](https://github.com/iBotPeaches/Apktool/pull/3865)] Add `-l/--lib` to specify shared libraries. (Thanks IgorEisberg)
- [[#3865](https://github.com/iBotPeaches/Apktool/pull/3865)] Fix support of shared libraries, parsing `pkgId` instead of guessing. (Thanks IgorEisberg)
- [[#3826](https://github.com/iBotPeaches/Apktool/pull/3826)] Fix escaping of CDATA like strings. (Thanks vbfox)
- [[#3849](https://github.com/iBotPeaches/Apktool/pull/3849)] Fix redundant flags in flag attrs. (Thanks IgorEisberg)
- [[#3838](https://github.com/iBotPeaches/Apktool/issues/3838)] Fix crash from ApkEditor "protect" mode.
- [[#3865](https://github.com/iBotPeaches/Apktool/pull/3865)] Change all tests to use aapt2 to phase out aapt1. (Thanks IgorEisberg)
- [[#3835](https://github.com/iBotPeaches/Apktool/pull/3835)] Upgrade `gradle/actions` to `4.3.1`.
- [[#3846](https://github.com/iBotPeaches/Apktool/pull/3846)] Upgrade `commons-io` to `2.19.0`.
- [[#3844](https://github.com/iBotPeaches/Apktool/pull/3844)] Upgrade `commons-text` to `1.13.1`.
- [[#3836](https://github.com/iBotPeaches/Apktool/pull/3836), [#3845](https://github.com/iBotPeaches/Apktool/pull/3845), [#3852](https://github.com/iBotPeaches/Apktool/pull/3852)] Upgrade `google/guava` to `33.4.7-jre`.
- [[#3884](https://github.com/iBotPeaches/Apktool/pull/3884), [#3897](https://github.com/iBotPeaches/Apktool/pull/3897)] Upgrade `xmlunit` to `2.10.3`.
